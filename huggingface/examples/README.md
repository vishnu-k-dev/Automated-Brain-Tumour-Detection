# Example Images

Place 4 sample MRI images here (one per class) for the Gradio demo:

- `glioma.jpg` — A glioma MRI sample
- `meningioma.jpg` — A meningioma MRI sample  
- `notumor.jpg` — A healthy brain MRI sample
- `pituitary.jpg` — A pituitary tumor MRI sample

Copy from your dataset:
```bash
cp src/data/raw/Testing/glioma/Te-gl_0010.jpg huggingface/examples/glioma.jpg
cp src/data/raw/Testing/meningioma/Te-me_0010.jpg huggingface/examples/meningioma.jpg
cp src/data/raw/Testing/notumor/Te-no_0010.jpg huggingface/examples/notumor.jpg
cp src/data/raw/Testing/pituitary/Te-pi_0010.jpg huggingface/examples/pituitary.jpg
```
